### 실행 방법

(pnp mode이므로 따로 yarn install 혹은 npm install을 할 필요가 없습니다!)

```
$ git clone https://github.com/Jay-WKJun/classTing_work

$ cd classTing_work

// yarn이 이미 있다면 생략가능
$ npm install -g yarn

$ yarn set berry

$ yarn dev

-> http://localhost:5173/ 에 접속!
```
